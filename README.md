# Annapurna

**Annapurna** is a full-stack, AI-powered recipe platform designed to help users cook smarter using the ingredients they already have. By combining image recognition, generative AI, and a scalable modern tech stack, Annapurna delivers personalized recipes, global cuisine discovery, and intelligent cooking assistance in a production-ready SaaS application.

---

## Key Features & Functionalities

### AI Pantry Scanner (My Pantry)
- Scan an image of pantry items using AI.
- Automatically detect available ingredients.
- Generate customized recipes based on detected items.
- Smart ingredient substitutions when something is missing.

### AI-Generated Recipes
Each recipe includes:
- Prep time and cook time  
- Servings information  
- Complete ingredient list  
- Step-by-step cooking instructions  
- Nutritional breakdown per serving  
- Chef’s tips and tricks  
- Intelligent ingredient substitutions  

### Global Recipe Discovery
- Browse a large recipe library organized by:
  - Cuisines from different countries
  - Moods and categories
- Easy navigation to find recipes based on cravings.

### Intelligent “How to Cook” Search
- Search for any recipe worldwide.
- AI chef generates detailed cooking instructions instantly.

### Save, Export & Personalize
- Save favorite recipes to personalized collections.
- Export recipes as PDF for offline access.
- Access saved recipes anytime from the dashboard.

## Pricing & Subscription Model

### Free Plan – $0/month
- 10 AI pantry scans
- 5 AI recipe generations
- Access to standard recipes

### Pro Plan – $7.99/month
- Unlimited pantry scans
- Unlimited AI recipe generation
- Advanced recipe insights (nutrition, substitutions, tips)

This subscription model makes Annapurna a real, monetizable SaaS product.

  - Cuisines
